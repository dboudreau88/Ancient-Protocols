# LPD LPR

## Overview

Line Printer Daemon/Remote. Berkeley Unix printing protocol. Client submits print jobs via LPR, server handles them via LPD.

## Era

1983

## What Makes It Interesting

Simple queue-based model. Control file + data file separation. Named print queues. Still supported by CUPS. Being removed from FreeBSD in 2026.

## Revival Potential

**Medium**

## Current Status

Declining. Still in CUPS. Being removed from FreeBSD (2026). Legacy but functional.
