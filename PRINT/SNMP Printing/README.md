# SNMP Printing

## Overview

Using SNMP (Simple Network Management Protocol) to query printer status and manage network printers.

## Era

1990s

## What Makes It Interesting

Standardized printer monitoring. Supply levels, page counts, error states. Printer MIB (RFC 3805) defines standard objects. Still used alongside IPP.

## Revival Potential

**Medium**

## Current Status

Alive. Still used for printer monitoring and management.
