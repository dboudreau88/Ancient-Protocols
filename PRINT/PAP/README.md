# PAP

## Overview

Printer Access Protocol. AppleTalk protocol for printer communication and sharing.

## Era

1980s

## What Makes It Interesting

Printer discovery was automatic. Status messages sent to user's screen. Bidirectional communication. Integrated with AppleTalk's zero-config networking.

## Revival Potential

**Low**

## Current Status

Dead. Died with AppleTalk.
