# AppSocket

## Overview

Raw TCP port 9100 printing. Also known as JetDirect, RAW, or Windows TCPmon. Developed by Tektronix.

## Era

1990s

## What Makes It Interesting

Dead simple: send data to port 9100. No handshake, no negotiation. The simplest, fastest, and most reliable network printing protocol. No security whatsoever.

## Revival Potential

**High**

## Current Status

Alive. Still the most common network printing method. Universally supported.
