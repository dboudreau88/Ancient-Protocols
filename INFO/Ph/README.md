# Ph

## Overview

Phone directory lookup protocol from Apollo (later HP). Federated directory service for looking up user contact information across networked systems.

## Era

1980s

## What Makes It Interesting

Federated directory service before LDAP. Simple presence/directory concept. Users could update their own entries. Preceded modern corporate directories.

## Revival Potential

**Low**

## Current Status

Dead. Concept absorbed by LDAP and modern directory services.
