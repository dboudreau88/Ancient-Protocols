# NNTP

## Overview

Network News Transfer Protocol. Powers Usenet newsgroups - the original distributed discussion system. Federated across thousands of news servers worldwide.

## Era

1980s-ongoing

## What Makes It Interesting

Federated moderation. Threaded discussions before forums existed. Binary groups for file sharing. Hierarchical naming (comp.*, rec.*, etc.). A decentralized social network decades before the term existed.

## Revival Potential

**Medium**

## Current Status

Alive (declining). Still used for binary file sharing. Text groups declining but functional. Many archives of historical value.
