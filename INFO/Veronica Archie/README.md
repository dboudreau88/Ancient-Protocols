# Veronica Archie

## Overview

Search engines for Gopher (Veronica) and FTP (Archie). Pre-Google search tools that indexed content on early internet services.

## Era

1990s

## What Makes It Interesting

Archie indexed FTP archives. Veronica searched Gopher menus. Subject-based indexing. Pre-Google search concepts that proved indexed search was viable. The ancestors of all search engines.

## Revival Potential

**Low**

## Current Status

Dead. Concepts live on in modern search engines. Veronica-2 still indexes Gopherspace.
