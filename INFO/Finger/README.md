# Finger

## Overview

User status/presence information protocol. Runs on TCP port 79. Users could place .plan and .project files in their home directory for others to read.

## Era

1977

## What Makes It Interesting

The original 'presence' protocol. .plan files were personal blogs before blogs existed. Simple and human-friendly. 'Fingering' someone showed their login status, office location, and custom message. Precursor to all modern presence systems.

## Revival Potential

**High**

## Current Status

Mostly dead but experiencing renewed interest. Hackaday covered it in 2025. 'Fingerverse' concept emerging. Anti-tracking presence alternative.
