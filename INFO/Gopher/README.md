# Gopher

## Overview

Menu-based document retrieval protocol. Pre-web competitor to HTTP. Created at University of Minnesota. Organizes content in hierarchical menus rather than linked pages.

## Era

1991-ongoing

## What Makes It Interesting

Structured navigation. No HTML bloat. Server-defined menus. Intentionally simple. Gemini is its modern spiritual successor. ~160 active servers indexed by Veronica-2. The Overbite Project maintains modern clients.

## Revival Potential

**High**

## Current Status

Alive (small). ~160 active servers. Overbite Project makes browser extensions. Growing community of enthusiasts.
