# Gemini

## Overview

Modern Gopher successor. TLS by default, simplified HTTP-like protocol. Created by Solderpunk as a response to web complexity.

## Era

2019-present

## What Makes It Interesting

Intentionally limited design. No JavaScript, no cookies, no tracking. TLS mandatory. Gemini pages (called 'capsules') use a simple markup language called Gemini text. Growing community. Proves the Gopher model works with modern security.

## Revival Potential

**High**

## Current Status

Alive and growing. Active community. Multiple clients and servers. Capsule directories emerging.
