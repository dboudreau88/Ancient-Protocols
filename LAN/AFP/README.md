# AFP

## Overview

Apple Filing Protocol. Mac-native file sharing protocol. Part of AppleTalk suite. Supported resource forks and Mac-specific metadata.

## Era

1985-2025

## What Makes It Interesting

Preserved resource forks and Mac metadata. Automatic service discovery via AppleTalk. AFP 3.0+ used TCP/IP. Handled Mac file system quirks that other protocols couldn't.

## Revival Potential

**Low**

## Current Status

Dying. Being removed from macOS 27. Superseded by SMB3.
