# LAT

## Overview

DEC Local Area Transport. Terminal server protocol for connecting terminals to VAX/VMS systems over Ethernet.

## Era

1983

## What Makes It Interesting

Terminal server protocol. Non-routable but efficient for local clusters. DECserver hardware ran LAT. Enabled centralized terminal access.

## Revival Potential

**Low**

## Current Status

Dead. Legacy DEC environments only.
