# IPX SPX

## Overview

Novell's network layer protocol suite. Internetwork Packet Exchange (IPX) for routing, Sequenced Packet Exchange (SPX) for reliable delivery.

## Era

1980s-2000s

## What Makes It Interesting

Simple addressing. Broadcast-based service discovery. Dominated LAN gaming - Doom, Quake, Diablo all used IPX. Still used in some industrial systems.

## Revival Potential

**Medium**

## Current Status

Mostly dead. Gaming nostalgia keeps interest alive. Some industrial use remains.
