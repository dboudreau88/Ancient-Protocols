# SNA

## Overview

IBM Systems Network Architecture. Hierarchical networking protocol suite for IBM mainframe environments.

## Era

1974-present

## What Makes It Interesting

Hierarchical networking. Still in mainframe environments. MATIP (Mapping of Airline Traffic over Internet Protocol) carries SNA over TCP/IP.

## Revival Potential

**Low**

## Current Status

Alive (mainframes). IBM mainframe environments still use it.
