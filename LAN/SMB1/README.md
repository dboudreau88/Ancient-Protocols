# SMB1

## Overview

Original Server Message Block protocol. IBM developed it for DOS-based LAN file sharing.

## Era

1983

## What Makes It Interesting

Simple file sharing. Cross-platform via Samba (reverse-engineered). Foundation of Windows file sharing. SMBv1 is insecure but historically important.

## Revival Potential

**Medium**

## Current Status

Superseded by SMBv2/v3. Historical interest. Samba still supports it for legacy compatibility.
