# DECnet

## Overview

Digital Equipment Corporation's networking protocol. Phase IV was the most widely deployed version.

## Era

1975-2000s

## What Makes It Interesting

Phase IV routing was elegant. DNA (DECnet Architecture) was comprehensive. Supported both DEC and non-DEC equipment. Hierarchical addressing.

## Revival Potential

**Low**

## Current Status

Dead. DEC is gone. Concepts absorbed into OSI.
