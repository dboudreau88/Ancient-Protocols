# 9P

## Overview

Plan 9's distributed file protocol. Everything is a file - network resources, services, and devices are all accessed through a unified namespace.

## Era

1992

## What Makes It Interesting

Everything is a file. Namespace-based resource access. Mount remote resources into your local file tree. Used in Linux (virtio-9p) for VM file sharing. Modern IoT relevance.

## Revival Potential

**Medium**

## Current Status

Alive (niche). Used in Linux VMs. Interesting for IoT and distributed systems.
