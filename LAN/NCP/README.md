# NCP

## Overview

Novell NetWare Core Protocol. File sharing and network services protocol for NetWare networks.

## Era

1983

## What Makes It Interesting

Fast for its era. Directory services (NDS/eDirectory). Login scripts. Bindery for simple directory. NetWare dominated LAN networking before Windows NT.

## Revival Potential

**Low**

## Current Status

Dead. NetWare is dead. Novell OES kept it alive until 2009.
