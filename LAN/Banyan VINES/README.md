# Banyan VINES

## Overview

Virtual Network Operating System. Distributed network operating system with StreetTalk global directory.

## Era

1984-1999

## What Makes It Interesting

StreetTalk directory was global and hierarchical - advanced for its time. Could name users by organization/user rather than machine. Advanced for its era but lost to Microsoft.

## Revival Potential

**Low**

## Current Status

Dead. Company is gone.
