# MUTE

## Overview

Anonymous file sharing via encrypted P2P routing. Messages routed through intermediate nodes to hide source and destination.

## Era

2002-present

## What Makes It Interesting

Anonymity through encrypted routing - each hop only knows the previous and next node. Concept similar to Tor but for file sharing. Decentralized approach to anonymous communication.

## Revival Potential

**Medium**

## Current Status

Dying. Niche but concept is ahead of its time.
