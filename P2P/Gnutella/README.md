# Gnutella

## Overview

Fully decentralized peer-to-peer file sharing protocol. No central server - uses query flooding to discover content. Created by Justin Frankel and Tom Pepper at Nullsoft (Winamp creators).

## Era

2000-ongoing

## What Makes It Interesting

Anti-censorship by design - no single point of failure. Ultrapeer/leaf architecture for scalability. HTTP-based transfers. Pioneered the concept of fully distributed file sharing after Napster's legal shutdown proved centralization was a vulnerability.

## Revival Potential

**High**

## Current Status

Alive (small). Multiple clients still exist: LimeWire (legacy), BearShare, gtk-gnutella, WireShare. Community maintains protocol specifications.
