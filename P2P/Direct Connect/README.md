# Direct Connect

## Overview

Hub-based P2P protocol with minimum share requirements. Users must share a minimum amount of content to participate. Hub operators set rules.

## Era

2000-present

## What Makes It Interesting

Forced sharing ratio ensured network health. Hub-based communities with enforced rules created curated spaces. Text-based protocol (ASCII). ADC (Advanced DC) is an improved version with binary protocol and encryption.

## Revival Potential

**Medium**

## Current Status

Alive (small). DC++ and other clients still active. ADC protocol is the modern version.
