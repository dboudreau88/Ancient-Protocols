# Soulseek

## Overview

Music-focused P2P file sharing with persistent chat rooms. Niche community centered around music discovery and sharing.

## Era

2000-present

## What Makes It Interesting

Search by directory browsing (not just filenames). Chat rooms organized by music genre. Community-driven curation. Still the best place to find rare/niche music. Nicotine+ is an active open-source client.

## Revival Potential

**High**

## Current Status

Alive with cult following. Official client and Nicotine+ actively maintained. Protocol documentation compiled by community.
