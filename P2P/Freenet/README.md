# Freenet

## Overview

Anonymous, censorship-resistant distributed datastore. Stores encrypted data on participants without their knowledge. Created by Ian Clarke.

## Era

2000-present

## What Makes It Interesting

Stores data without knowing what it is - plausible deniability by design. Darknet mode for friend-to-friend only. Content is distributed and replicated across the network. Higher security = slower performance.

## Revival Potential

**High**

## Current Status

Alive. Actively developed. Privacy focus more relevant than ever.
