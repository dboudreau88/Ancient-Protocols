# WinMX

## Overview

P2P file sharing using WPNP (WinMX Peer Network Protocol) and OpenNAP protocols. Dual-network design for redundancy.

## Era

2000-2005

## What Makes It Interesting

Dual-network design was clever - could connect via either protocol. Peer caching helped with NAT traversal. Community-run servers kept it alive after official shutdown.

## Revival Potential

**Low**

## Current Status

Dying. Community-run servers exist but user base is tiny.
