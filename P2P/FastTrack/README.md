# FastTrack

## Overview

Semi-decentralized P2P protocol used by Kazaa, Grokster, and iMesh. Introduced the 'supernode' concept where powerful users become indexing hubs for others.

## Era

2001

## What Makes It Interesting

Supernode architecture was elegant - powerful machines naturally became search hubs, creating a hierarchical overlay network. Balanced decentralization with efficiency. Kazaa had 389M+ downloads at peak.

## Revival Potential

**Medium**
