# Napster

## Overview

First mass-market P2P file sharing service. Centralized index server with peer-to-peer transfers. Created by Shawn Fanning.

## Era

1999-2001

## What Makes It Interesting

Changed music industry forever. 80M users at peak. Central index made it easy to search but also easy to shut down. OpenNAP protocol variant spawned communities like WinMX that survived Napster's death.

## Revival Potential

**Medium**

## Current Status

Dead. OpenNAP protocol variant still has community (WinMX). Federated Napster concept lives on in protocols like Soulseek.
