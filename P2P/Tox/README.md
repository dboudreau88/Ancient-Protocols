# Tox

## Overview

Encrypted P2P messenger using distributed hash table (DHT). No servers, no phone number, no registration required.

## Era

2013-present

## What Makes It Interesting

No servers at all - fully decentralized. End-to-end encrypted by default. No registration needed - generate a Tox ID and start messaging. Audio/video calling. Group chats.

## Revival Potential

**High**

## Current Status

Alive. Multiple clients: qTox, Toxic, TRIfA (Android). Active development.
