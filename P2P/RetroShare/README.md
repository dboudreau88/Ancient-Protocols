# RetroShare

## Overview

Friend-to-friend encrypted P2P platform. Only connects to people you explicitly trust. Built-in chat, forums, file sharing, and channel system.

## Era

2006-present

## What Makes It Interesting

Trust graph determines connectivity. Darknet by default (can't discover random users). Full-featured communication platform. End-to-end encrypted. Built-in forums and channels create a private social network.

## Revival Potential

**High**

## Current Status

Alive and actively developed. Scored 6/7 on EFF secure messaging scorecard.
