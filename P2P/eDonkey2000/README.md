# eDonkey2000

## Overview

Hash-based file sharing using multipart chunked downloads. Content-addressed before it was cool. Client-server model with server network for indexing.

## Era

2000-ongoing

## What Makes It Interesting

Content-addressed file sharing (hash-based) was ahead of its time - same concept used in BitTorrent and IPFS. Credit system for uploaders encouraged sharing. Could download same file from multiple sources simultaneously.

## Revival Potential

**Medium**

## Current Status

Alive (small). MLDonkey and eMule clients still running. Community maintained.
