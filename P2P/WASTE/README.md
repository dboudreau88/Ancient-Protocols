# WASTE

## Overview

Justin Frankel's (Winamp/Gnutella creator) encrypted darknet tool. Created small trusted networks over the internet.

## Era

2003-2005

## What Makes It Interesting

Created by a legend of P2P. Encrypted mesh between trusted friends. Minimalist design. Could create private networks without exposing them to the internet.

## Revival Potential

**Medium**

## Current Status

Dead. Concept needs modern reimplementation with current crypto standards.
