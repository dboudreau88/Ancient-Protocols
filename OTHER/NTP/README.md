# NTP

## Overview

Network Time Protocol. Distributed clock synchronization protocol. Keeps computer clocks accurate across the internet.

## Era

1985

## What Makes It Interesting

Distributed clock synchronization. Byzantine fault tolerant. Stratified time sources. Still essential for internet infrastructure. Every connected device uses it.

## Revival Potential

**High**

## Current Status

Alive. Still essential. PTP (IEEE 1588) is the modern high-precision version.
