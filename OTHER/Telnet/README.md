# Telnet

## Overview

Remote terminal access protocol. The original way to log into remote computers over a network.

## Era

1969

## What Makes It Interesting

Simplest remote login. Works everywhere. Foundation of remote access. Replaced by SSH for security, but the concept is timeless. Still used for device management.

## Revival Potential

**Medium**

## Current Status

Alive (legacy). Replaced by SSH for security. Still used for network device management and BBS access.
