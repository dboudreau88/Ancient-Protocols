# NFS

## Overview

Network File System (Sun Microsystems). Allows mounting remote filesystems as local drives.

## Era

1984

## What Makes It Interesting

Mount remote filesystems locally. Stateless protocol (v2/v3). Unix standard for network file sharing. Still essential in Linux/Unix environments.

## Revival Potential

**High**

## Current Status

Alive. Still widely used in Unix/Linux. NFSv4 is current version.
