# RADIUS

## Overview

Remote Authentication Dial-In User Service. AAA (Authentication, Authorization, Accounting) protocol. Centralized auth for ISPs and enterprise networks.

## Era

1997

## What Makes It Interesting

AAA protocol. Centralized authentication. Accounting for usage tracking. Widely used by ISPs, VPNs, and enterprise networks. UDP-based with shared secret.

## Revival Potential

**High**

## Current Status

Alive. Still widely used. Diameter is the modern successor but RADIUS persists.
