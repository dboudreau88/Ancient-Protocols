# WHOIS

## Overview

Domain name and contact information lookup protocol. Simple text-based query/response.

## Era

1982

## What Makes It Interesting

Simple text-based query. Decentralized registrars. Whois servers for each TLD. Still used for domain ownership lookup. Being replaced by RDAP for structured data.

## Revival Potential

**Medium**

## Current Status

Alive (declining). Still used but being replaced by RDAP.
