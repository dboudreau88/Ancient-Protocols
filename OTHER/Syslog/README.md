# Syslog

## Overview

System logging protocol (RFC 3164/5424). UDP-based log collection and forwarding.

## Era

2001

## What Makes It Interesting

UDP-based log collection. Simple and universal. Structured data in RFC 5424. Every Unix system uses it. Foundation of centralized logging (ELK, Splunk, etc.).

## Revival Potential

**High**

## Current Status

Alive. Essential for logging infrastructure. RFC 5424 is the current standard.
