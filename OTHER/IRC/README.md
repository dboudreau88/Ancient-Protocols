# IRC

## Overview

Internet Relay Chat. Channel-based real-time text chat. Federated servers. One of the oldest still-active chat systems.

## Era

1988

## What Makes It Interesting

Channel-based real-time chat. Federated servers. Simple text protocol. Still used by open source projects, gaming communities, and privacy-conscious users. Inspired Slack, Discord, and modern chat apps.

## Revival Potential

**High**

## Current Status

Alive. Still used. Inspires modern chat applications. Many clients and servers exist.
