# AFS

## Overview

Andrew File System (Carnegie Mellon). Large-scale distributed filesystem with client-side caching.

## Era

1984

## What Makes It Interesting

Large-scale distributed filesystem. Client-side caching for performance. Kerberos authentication. Cell-based naming. Used in universities worldwide.

## Revival Potential

**Medium**

## Current Status

Alive (niche). Used in academia. OpenAFS is the open-source implementation.
