# SNMP

## Overview

Simple Network Management Protocol. UDP-based protocol for network monitoring and management.

## Era

1988

## What Makes It Interesting

UDP-based network monitoring. MIB (Management Information Base) trees. Get/Set/Trap operations. Still dominant for network device management. v3 added security.

## Revival Potential

**High**

## Current Status

Alive. Still dominant for network management. v3 is current version.
