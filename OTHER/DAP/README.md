# DAP

## Overview

OSI Directory Access Protocol. Precursor to LDAP. Part of X.500 directory services standard.

## Era

1981

## What Makes It Interesting

Precursor to LDAP. X.500 directory services. Hierarchical directory structure. Complex but comprehensive. Too heavy for most use cases, which is why LDAP won.

## Revival Potential

**Low**

## Current Status

Dead. LDAP won completely.
