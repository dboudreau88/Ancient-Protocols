# LDAP

## Overview

Lightweight Directory Access Protocol. Hierarchical directory services for looking up people, devices, and resources.

## Era

1993

## What Makes It Interesting

Hierarchical directory services. Authentication backend (Active Directory, OpenLDAP). Used for SSO, address books, and device directories. Foundation of enterprise identity management.

## Revival Potential

**High**

## Current Status

Alive. Essential for enterprise identity. Active Directory and OpenLDAP are widespread.
