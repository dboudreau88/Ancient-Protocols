# Rlogin

## Overview

Remote login protocol. Simpler than Telnet. Trusted host authentication via rhosts files.

## Era

1981

## What Makes It Interesting

Simpler than Telnet. rhosts-based authentication (trusted hosts). No negotiation - just connect and get a shell. Clean and minimal.

## Revival Potential

**Low**

## Current Status

Dead. Security issues with rhosts authentication.
