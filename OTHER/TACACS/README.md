# TACACS

## Overview

Terminal Access Controller Access Control System. Centralized authentication for network devices (routers, switches).

## Era

1983

## What Makes It Interesting

Centralized authentication for network devices. Predecessor to TACACS+ and RADIUS. Used in early Cisco environments.

## Revival Potential

**Low**

## Current Status

Legacy. Replaced by TACACS+ and RADIUS.
