# GameSpy

## Overview

GameSpy master server protocol for game matchmaking and server browsers.

## Era

1999-2013

## What Makes It Interesting

Centralized server browser. NAT traversal helpers. User profiles and statistics. The backbone of online PC gaming for a decade.

## Revival Potential

**Low**

## Current Status

Dead. GameSpy shut down in 2013. Community replacements exist (OpenSpy, GameRanger).
