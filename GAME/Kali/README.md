# Kali

## Overview

Tunneling IPX over TCP/IP. Enabled LAN-only games to be played over the internet by wrapping IPX packets in TCP/IP.

## Era

1990s-2000s

## What Makes It Interesting

Clever protocol translation. Enabled LAN games online before game developers added internet multiplayer. The bridge between LAN gaming and online gaming.

## Revival Potential

**Medium**

## Current Status

Mostly dead. Concept useful for retro gaming.
