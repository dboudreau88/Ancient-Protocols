# IPX SPX Gaming

## Overview

IPX/SPX used for LAN gaming. Doom, Quake, Diablo, StarCraft, and many other games used IPX for multiplayer.

## Era

1990s

## What Makes It Interesting

Low-latency LAN gaming. Broadcast-based server discovery. Games like Doom could be played over IPX with zero configuration. The golden age of LAN parties.

## Revival Potential

**Medium**

## Current Status

Mostly dead. Retro gaming community keeps it alive via IPX wrappers and emulators like DOSBox.
