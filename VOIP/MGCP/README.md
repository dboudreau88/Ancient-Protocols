# MGCP

## Overview

Media Gateway Control Protocol. Centralized call control protocol for media gateways. Decomposes gateway intelligence into call agents and media gateways.

## Era

1999

## What Makes It Interesting

Decomposes gateway intelligence. Useful for carrier-grade deployments. Call agent controls multiple gateways. Used in large VoIP infrastructure deployments.

## Revival Potential

**Low**

## Current Status

Alive (carriers). Specialized use case in telecom infrastructure.
