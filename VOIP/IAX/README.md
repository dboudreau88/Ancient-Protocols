# IAX

## Overview

Inter-Asterisk Exchange. VoIP protocol designed for the Asterisk PBX. Uses a single UDP port (4569) for both signaling and media, unlike SIP which requires multiple ports.

## Era

2004

## What Makes It Interesting

NAT-friendly by design - single port for everything. Trunking between Asterisk servers. Simpler than SIP for many use cases. Better performance on lossy networks due to IAX2's trunking feature.

## Revival Potential

**High**

## Current Status

Alive (niche). Still used in Asterisk. Simpler than SIP for many deployments.
