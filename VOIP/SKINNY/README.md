# SKINNY

## Overview

Cisco Skinny Client Control Protocol (SCCP). Lightweight protocol for controlling Cisco IP phones from Cisco Unified Communications Manager.

## Era

2000s

## What Makes It Interesting

Simple station control. Less overhead than SIP for Cisco environments. Efficient for managed enterprise networks where Cisco infrastructure is dominant.

## Revival Potential

**Low**

## Current Status

Alive (Cisco). Cisco-locked. Used in Cisco UC environments.
