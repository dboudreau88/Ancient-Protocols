# Jingle

## Overview

XMPP extension for peer-to-peer VoIP/video signaling. Designed by Google and the XMPP Standards Foundation. Uses ICE for NAT traversal.

## Era

2005

## What Makes It Interesting

Google Talk used it. Open standard. Uses ICE for NAT traversal. libjingle library released under BSD license. Supports audio, video, and file transfer sessions.

## Revival Potential

**Medium**

## Current Status

Alive (niche). libjingle is open source. Underused despite being well-designed.
