# H.323

## Overview

ITU VoIP standard. Predecessor to SIP. Comprehensive protocol suite for multimedia communication over IP networks.

## Era

1996

## What Makes It Interesting

Battle-tested in enterprise and carrier environments. Still used in some video conferencing systems. Well-understood and well-documented. Supports gatekeepers for call routing and bandwidth management.

## Revival Potential

**Low**

## Current Status

Declining. Superseded by SIP in most deployments. Still found in legacy enterprise systems.
