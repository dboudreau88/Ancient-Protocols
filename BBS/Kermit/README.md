# Kermit

## Overview

Transport-independent file transfer and terminal emulation protocol. Created at Columbia University. Works over serial, TCP/IP, X.25, modem, and other connections.

## Era

1981

## What Makes It Interesting

Works over literally anything. Built-in scripting language for automation. E-Kermit for embedded devices. Can run over SSH/SSL/TLS. Still maintained. Named after the Muppet.

## Revival Potential

**High**

## Current Status

Alive. Kermit Project continues at kermitproject.org. Open source. Used in embedded devices.
