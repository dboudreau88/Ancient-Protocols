# ANSI VT100

## Overview

Terminal escape code protocols for text-based user interfaces. VT100 (DEC) and ANSI escape sequences enabled color graphics, cursor movement, and screen clearing over text terminals.

## Era

1970s-present

## What Makes It Interesting

BBS art culture thrived on ANSI escape codes. Color graphics over text. Still used in every terminal emulator today. The foundation of all terminal-based UIs.

## Revival Potential

**Medium**

## Current Status

Alive. Every terminal emulator in existence supports these codes.
