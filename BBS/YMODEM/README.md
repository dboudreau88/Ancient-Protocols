# YMODEM

## Overview

XMODEM successor by Chuck Forsberg. 1KB blocks, batch file transfer support, CRC-16 error checking, and filename/size header packet.

## Era

1985

## What Makes It Interesting

Batch file transfer was a big improvement. Header packet with metadata. Better performance than XMODEM while maintaining compatibility.

## Revival Potential

**Medium**

## Current Status

Alive (retro). Better than XMODEM for modern retro use.
