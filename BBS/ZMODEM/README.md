# ZMODEM

## Overview

Serial file transfer protocol developed by Chuck Forsberg. Featured restartable transfers, auto-start by sender, 32-bit CRC, and control character quoting for 8-bit clean transfers.

## Era

1986

## What Makes It Interesting

Resume interrupted transfers was revolutionary. 8KB blocks for performance. Crash recovery. Dominated BBS file transfers in the early 1990s. Became the standard as widespread as XMODEM before it.

## Revival Potential

**High**

## Current Status

Alive in retro-computing community. WiModem232 uses it. Still the go-to for serial file transfer.
