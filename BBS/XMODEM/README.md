# XMODEM

## Overview

Original serial file transfer protocol by Ward Christensen. 128-byte blocks with simple checksum. The 'hello world' of file transfer.

## Era

1977

## What Makes It Interesting

Maximum compatibility - works on virtually anything with a serial connection. Simple enough to implement in a few hundred lines of code. Created the template for all serial file transfer protocols.

## Revival Potential

**Medium**

## Current Status

Alive (retro). Maximum compatibility for vintage hardware.
