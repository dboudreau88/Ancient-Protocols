# BBS Door Protocols

## Overview

Protocols used by BBS door games and applications (Synchronet, Renegade, Mystic BBS). Doors were external programs that BBS sysops could install for users to play games or use utilities.

## Era

1980s-1990s

## What Makes It Interesting

Doors created a platform-within-a-platform ecosystem. Multiplayer BBS games were the original online gaming. Door drop files standardized how games interfaced with BBS software.

## Revival Potential

**Medium**

## Current Status

Alive (retro). Active BBS revival scene. Telnet BBS Guide lists active BBSes.
