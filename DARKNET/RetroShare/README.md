# RetroShare

## Overview

Friend-to-friend encrypted P2P platform. Only connects to people you explicitly trust via cryptographic certificates.

## Era

2006-present

## What Makes It Interesting

Trust graph determines connectivity. Darknet by default. Full-featured: chat, forums, file sharing, channels. End-to-end encrypted. Built-in web server for forums. Scored 6/7 on EFF scorecard.

## Revival Potential

**High**

## Current Status

Alive and actively developed. Cross-platform (Linux, Windows, macOS, Android).
