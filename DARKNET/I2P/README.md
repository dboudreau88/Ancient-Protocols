# I2P

## Overview

Invisible Internet Project. Anonymous overlay network using garlic routing (similar to onion routing but bundles messages).

## Era

2003-present

## What Makes It Interesting

Garlic routing. Hidden services ('eepsites'). Tunnel-based architecture. Unidirectional tunnels for better anonymity. Used for dark web services. Active development.

## Revival Potential

**High**

## Current Status

Alive. Active development. Used by privacy-conscious users. Growing ecosystem.
