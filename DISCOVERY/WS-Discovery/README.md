# WS-Discovery

## Overview

Web Services Discovery. XML-based device announcement protocol for SOAP web services.

## Era

2004

## What Makes It Interesting

SOAP-based discovery for enterprise. WS-Management uses it. Designed for complex enterprise environments.

## Revival Potential

**Low**

## Current Status

Declining. Overly complex. mDNS won for most use cases.
