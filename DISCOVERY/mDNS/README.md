# mDNS

## Overview

Multicast DNS + DNS Service Discovery (DNS-SD). Zero-configuration networking. Apple's implementation is called Bonjour; Linux uses Avahi.

## Era

1999-present

## What Makes It Interesting

Plug-and-play device discovery. Works without infrastructure. Devices announce themselves on the local network. Used for printers, file shares, media devices, and IoT. Essential for modern smart homes.

## Revival Potential

**High**

## Current Status

Alive and thriving. IoT essential. mDNS (RFC 6762) and DNS-SD (RFC 6763) are active standards.
