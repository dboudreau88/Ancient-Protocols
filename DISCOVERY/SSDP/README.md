# SSDP

## Overview

Simple Service Discovery Protocol. Part of UPnP. Used by home routers, media devices, and IoT for auto-discovery.

## Era

1999

## What Makes It Interesting

Auto-discovery of UPnP devices. UDP-based. Used by Windows for network device discovery. Home routers use it for UPnP port forwarding.

## Revival Potential

**Medium**

## Current Status

Alive. Still in use for home routers and media devices.
