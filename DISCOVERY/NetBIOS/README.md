# NetBIOS

## Overview

Network Basic Input/Output System. Name resolution and session service for LAN communication.

## Era

1983

## What Makes It Interesting

Simple name-based addressing. Broadcast discovery. Foundation of Windows file sharing. Could resolve names to IP addresses on local networks.

## Revival Potential

**Low**

## Current Status

Declining. Legacy Windows networking. Replaced by DNS-based resolution.
