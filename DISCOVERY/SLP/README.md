# SLP

## Overview

Service Location Protocol. IETF standard for LAN service discovery without central directory infrastructure.

## Era

1997

## What Makes It Interesting

Open standard (RFC 2608). Scopes for organizing services. Dynamic registration and deregistration. Agent and directory agent architecture. Simpler than mDNS in some ways.

## Revival Potential

**Medium**

## Current Status

Alive (niche). Alternative to mDNS/Bonjour. Used in some enterprise environments.
