# PowWow

## Overview

Tribal Voice's PowWow instant messaging client. Competed with AIM in the late 1990s before being crushed by AOL's dominance.

## Era

1990s

## What Makes It Interesting

Early voice chat integration. Room-based chatting predating modern group chat. Was a legitimate competitor to AIM before AOL's network effects won.

## Revival Potential

**Low**

## Current Status

Dead. No documentation or protocol specs exist.
