# TOC

## Overview

Text-based gateway protocol to OSCAR. Provided a simpler, text-only alternative for lighter AIM clients that didn't need the full binary OSCAR protocol.

## Era

1990s

## What Makes It Interesting

Easy to reimplement. Text-based design made it accessible for CLI-first or embedded messaging clients. Good entry point for understanding OSCAR concepts.

## Revival Potential

**Medium**

## Current Status

Dead. Superseded by OSCAR and later XMPP.
