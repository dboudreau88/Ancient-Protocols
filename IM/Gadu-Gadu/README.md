# Gadu-Gadu

## Overview

Polish instant messaging protocol and client. Hugely popular in Poland, largely unknown elsewhere. Simple binary protocol.

## Era

2000-present

## What Makes It Interesting

Fascinating case study of regional protocol survival. While the rest of the world moved to proprietary apps, Gadu-Gadu persisted in Poland due to cultural inertia and local network effects.

## Revival Potential

**Low**

## Current Status

Alive but declining. Still used in Poland.
