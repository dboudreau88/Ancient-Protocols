# ICQ

## Overview

Original ICQ instant messenger protocol. ICQ was the first widely popular standalone IM client, created by Mirabilis (Israel). Used UIN (Universal Internet Number) as user identifiers.

## Era

1996-2024

## What Makes It Interesting

First widely-used IM protocol. UIN numbering system was novel - your identity was a number. Introduced concepts like 'online/offline' status that became universal. 'Uh oh!' notification sound was iconic.

## Revival Potential

**Low**

## Current Status

Dead (2024). ICQ used OSCAR protocol after AOL acquisition. Historical significance as the first mainstream IM.
