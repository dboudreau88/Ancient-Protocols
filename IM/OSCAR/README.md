# OSCAR

## Overview

AOL's binary instant messaging protocol used by AIM and ICQ. Uses FLAP framing with SNAC message types organized into 'foodgroups' for different services (messaging, buddy lists, file transfer, presence). Binary protocol transmitted in network byte order.

## Era

1997-2017

## What Makes It Interesting

Rich feature set including buddy lists, file transfer, chat rooms, typing indicators, idle time detection, and 'eviling' (user reporting). Elegant layered architecture with FLAP/SNAC/TLV structure. Supported TLS between FLAP and TCP layers.

## Revival Potential

**High**

## Current Status

Dead (official). Community revival via NINA wiki (nina.chat) and Open OSCAR Server project. Full protocol documentation maintained by volunteers.
