# MSNP

## Overview

Microsoft Notification Protocol used by MSN Messenger and Windows Live Messenger. Text-based protocol with 'switchboard' sessions for multi-party chat.

## Era

1999-2012

## What Makes It Interesting

Clean text protocol design. Switchboard concept elegantly handled group chats by creating temporary session rooms. Supported custom emoticons, winks, and nudges.

## Revival Potential

**Medium**

## Current Status

Dead. Reverse-engineered by community; supported in Pidgin/libpurple.
