# XMPP

## Overview

Extensible Messaging and Presence Protocol (formerly Jabber). Open XML-based standard for presence and messaging. Federated like email - any server can talk to any other server.

## Era

1999-present

## What Makes It Interesting

Federation model means no single company controls it. Extensible via XEPs (XMPP Extension Protocols). Jingle extension adds VoIP/video. Google Talk was built on XMPP. Used by WhatsApp internally (early versions).

## Revival Potential

**High**

## Current Status

Alive but niche. Active development continues. Needs modern UX to compete with proprietary apps.
