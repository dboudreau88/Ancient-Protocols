# YMSG

## Overview

Yahoo Messenger protocol. Binary protocol supporting chat rooms, file transfer, voice chat, and webcam sharing.

## Era

1998-2016

## What Makes It Interesting

Legendary chat rooms with public/private messaging. Multi-party conferencing built in. Supported voice and video before most competitors.

## Revival Potential

**Medium**

## Current Status

Dead. Partially documented by reverse engineering efforts.
