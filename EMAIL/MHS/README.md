# MHS

## Overview

Message Handling System by Action Technologies, later promoted by Novell. Integrated with NetWare directory services.

## Era

1986

## What Makes It Interesting

Integrated with NetWare directory. Gateway to other email systems. Novell promoted it heavily but abandoned it after purchasing WordPerfect Office.

## Revival Potential

**Low**

## Current Status

Dead. Novell abandoned it.
