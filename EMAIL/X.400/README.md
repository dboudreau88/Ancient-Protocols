# X.400

## Overview

ITU/OSI email standard. Rich addressing system with O/R (Originator/Recipient) addresses. Mandated by governments under GOSIP.

## Era

1984

## What Makes It Interesting

Rich addressing (O/R addresses could include country, organization, name, etc.). Built-in security features. Mandated by governments. Lost to SMTP due to simplicity.

## Revival Potential

**Low**

## Current Status

Dying. Lost to SMTP. Still in some military/government systems.
