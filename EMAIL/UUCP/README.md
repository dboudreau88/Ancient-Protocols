# UUCP

## Overview

Unix-to-Unix Copy. Suite of programs for remote execution, file transfer, email, and netnews between computers. Originally designed for dial-up connections.

## Era

1979-present

## What Makes It Interesting

Delayed delivery model. Route-based addressing (host1!host2!user). Worked without persistent connections. Each computer had a list of neighbor systems with phone numbers. The foundation of early email routing.

## Revival Potential

**High**

## Current Status

Alive (very niche). Hackaday covered it in 2025. Great for low-connectivity scenarios. Taylor UUCP is open source.
