# MH

## Overview

Message Handler. Unix email user agent where each message is treated as a separate file. CLI-first design.

## Era

1977

## What Makes It Interesting

Each message is a file - composable with shell tools. CLI-first design. No monolithic mailbox file. Inspired modern CLI email tools like himalaya and mujmap. The Unix philosophy applied to email.

## Revival Potential

**Medium**

## Current Status

Alive (niche). Still maintained. Inspires modern CLI email tools.
