# POP2

## Overview

Post Office Protocol version 2. Simple mail retrieval protocol. Downloaded emails from server to local client.

## Era

1985

## What Makes It Interesting

Simpler than POP3. No permanent storage on server - downloaded and deleted. Designed for era when server storage was expensive. Minimal protocol overhead.

## Revival Potential

**Low**

## Current Status

Dead. POP3 superseded it completely.
